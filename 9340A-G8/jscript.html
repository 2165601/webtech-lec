Tim Berners-Lee


- Computer Scientist

- Is the inventor of the World Wide Web and he created a new computer language called HTML (Hypertext Markup Language) which most web pages are written in. The first web page was available on 6 August 1991.

- Berners-Lee now leads the World Wide Web Consortium. That is an organization that looks after the World Wide Web. He is the author of the book Weaving the Web. He is a director of The Web Science Research Initiative (WSRI), and a member of the advisory board of the MIT Center for Collective Intelligence. In April 2009, he was elected as a member of the United States National Academy of Sciences, based in Washington, D.C. In 1999, Time Magazine named Berners-Lee one of the 100 Most Important People of the 20th Century. In March 2000 he was awarded an honorary degree from the Open University as Doctor of the University.

What is HyperText Markup Language?



Hypertext markup language is a computer language that is used to create documents on the world wide web. It uses plain text format with the help uf markup tags


Standards and Documentation

Many standards have changed over the years.To keep track standards were published by WHATWG and the W3 consortium


HTML 2.0 - November 1995

HTML 2.0 changes did not make too much of major changes to how HTML works however it did change how information is submitted. HTML 2.0 introduced multiplexing which made the delivery of website content faster because multiple requests were allowed

Full HTML 2.0 specifications

HTML 3.2 - January 1997

HTML 3.2 superceeded HTML 3.0 because the implementation of HTML 3.0 is too difficult along with the unwillingness of proprietary vendors like netscape. Html 3.0 had the ability to code tables , display math functions/equations and allow text flow through objects. However the HTML 3.0 was drafted and 3.2 had the W3C reccomendation.
HTML 3.2 has:

  • HTML 2.0 with some functions removed
  • Features of HTML 3.0 with some features modified
  • Full Backwards compatibility with HTML 2.0
Full HTML 3.2 specifications

HTML 4.0/4.01- December 1997

In HTML 3.2, tags and attributes were used to style the pages one-by-one. This turned out to be a tiring task. So in HTML 4, They provided tags for stylesheets, scripts and objects. In HTML 4.01, they added and modified some funtionality of tags like "form" tag and added a document type definition namely "strict"
HTML 4.0/4.01 has:

  • Tags for stylesheets, scripts, embedded objects and frames
  • Added new elemets like ABBR, ACRONYM, BDO, BUTTON, COL, etc. read into this for more changes from HTML 3.2 to 4.0
  • Made the market of broswer vendors conform to HTML standards more
Full HTML 4.01 specifications
